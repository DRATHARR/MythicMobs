## Description
Checks whether the specified metaskill exists


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| name | n, skill, s | The metaskill to check the existance of |  |



## Examples
```yaml
  Conditions:
  - isSkill{skill=ExampleMetaSkill}
```
