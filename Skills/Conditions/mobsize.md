## Description
Checks the size of an entity that can have its size changed (Slimes, Magma_cubes, and Phantoms). 


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| size      | s         | The size range to check                                              |         |


## Examples
```yaml
- size{s=>5} true
```


## Aliases
- [x] size
