## Description
Targets any child entities summoned by the caster.  
Works for player casters, if the [summon] mechanic was used and the `summonerIsOwner` attribute was set to `true`.


## Attributes
>*This targeter has no attributes*


## Examples
```yaml
  Skills:
  - heal{a=10} @Children
```


## Aliases
- [x] child
- [x] summons
