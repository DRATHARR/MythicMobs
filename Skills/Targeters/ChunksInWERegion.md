## Description
Targets the 0,0 chunk corners of the chunks in a specified WorldGuard region


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| region    | r         | The name of the region                                               |         |


## Examples
```yaml
  Skills:
  - setchunkforceloaded @ChunksInWERegion{r=exampleregion}
```
