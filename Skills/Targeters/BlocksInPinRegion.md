## Description
Targets the blocks in a region delimited by two pins


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| pin1      | p1        | The first [pin]                                                      |<!--type:Pin--> |
| pin2      | p2        | The second [pin]                                                     |<!--type:Pin--> |


## Examples
```yaml
  - particle @BlocksInPinRegion{p1=mypin;p2=anotherpin}
```
