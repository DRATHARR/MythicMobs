## Description
Causes the caster to equip a copy of the target's equipment


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| slots     | s         | The [slots](/Mobs/Equipment#slot) to copy                            | ALL<!--type:EquipSlot-->|


## Examples
```yaml
  Skills:
  - equipcopy @target
```
