## Description
Causes the casting goat mob to ram the targeted entity

> **This is a [Paper-Only] mechanic!**


## Attributes
> *This mechanic has no attributes*


## Examples
```yaml
ExampleMob:
  Type: GOAT
  Skills:
  - goatram @trigger ~onInteract
```
