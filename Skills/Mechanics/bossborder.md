## Description
Traps the the target player within a vanilla world border around the caster.


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| radius    | r         | The radius of the border                                             | 32      |


## Examples
```yml
  Skills:
  - bossBorder{r=40} @NearestPlayer{r=30}
```


## Aliases
- [x] effect:bossBorder
- [x] e:bossBorder
