## Description
Sets the head pitch of the target entity


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| amount    | a, pitch, p | The value the pitch will be set to                                 | 0       |


## Examples
```yaml
  Skills:
  - setpitch{amount=45} @EIR{r=10}
```
