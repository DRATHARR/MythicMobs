## Description
Logs a message to console directly. For debugging or preservation purposes.  


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| message   | m         | The message to log                                                   | ""      |


## Examples
```yaml
  Skills:
  - log{message="Hello World!"} @None
```
