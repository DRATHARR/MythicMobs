## Description
Makes the player fly, similar to the /fly command. Can use any aura
attribute.


## Attributes
> This mechanic inherits every *inheritable* attribute of the [Aura](/Skills/Mechanics/Aura) mechanic

  
## Examples
```yaml
  Skills:
  - fly{duration=100} @trigger ~onInteract
```
