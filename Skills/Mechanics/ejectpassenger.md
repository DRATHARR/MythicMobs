## Description
Ejects any passengers that are currently riding the mob.   

> **This is a no-target mechanic, and the affected entity will always be the caster**


## Attributes
> *This mechanic has no attributes*


## Examples
```yaml
  Skills:
  - ejectpassenger ~onDamaged
```


## Aliases
- [x] eject_passenger
