## Description
Sets the force-loaded status of a location's chunk


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| loaded    | l         | Whether the chunk should be force loaded                             | true    |


## Examples
```yaml
  Skills:
  - setchunkforceloaded @selflocation
```
