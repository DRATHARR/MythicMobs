## Description
Sets the entity's [pose](https://jd.papermc.io/paper/1.21.6/org/bukkit/entity/Pose.html)


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| pose      | p         | The pose to assign                                    | <!--type:EntityPose--> |


## Examples
```yaml
  Skills:
  - setpose{pose=DYING}
```
