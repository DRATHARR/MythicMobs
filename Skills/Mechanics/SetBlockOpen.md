## Description
Sets the target block's open state


## Attributes
| Attribute | Aliases   | Description                                                          | Default |
|-----------|-----------|----------------------------------------------------------------------|---------|
| open      | o         | Whether the block should be set as open                              | true    |


## Examples
```yaml
  Skills:
  - setblockopen @ObstructingBlock
```
